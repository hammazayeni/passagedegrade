# Taekwondo Sbeitla App

This project is a React application built with Vite, TypeScript, and Tailwind CSS. It is designed to manage Taekwondo promotion tests.

## How to Run in Trae AI (or Locally)

1.  **Install Dependencies**:
    Open the terminal in Trae AI and run:
    ```bash
    npm install
    # or
    pnpm install
    # or
    yarn install
    ```

2.  **Start the Development Server**:
    ```bash
    npm run dev
    # or
    pnpm run dev
    # or
    yarn dev
    ```

3.  **Open the App**:
    Click the link shown in the terminal (usually `http://localhost:5173`) to view the application.

## Features

-   **Dashboard**: Manage students, import from CSV, set grades.
-   **Projection Mode**: A visual display for the exam day.

## Tech Stack

-   React
-   TypeScript
-   Vite
-   Tailwind CSS
-   Shadcn UI
